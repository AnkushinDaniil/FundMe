# FundMe
FundMe is a smart contract that allows funders to fund and owner to withdraw.


## Installation
```bash
git clone https://github.com/AnkushinDaniil/hardhat-smartcontract-lottery.git
yarn install
```

## Testing
localhost:
```bash
hh test
```
sepolia:
```bash
hh test --network sepolia
```

## Node
localhost:
```bash
hh node
```
